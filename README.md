# Pygments lexer for Vehicle
A simple lexer that highlights vehicle code. To install, make sure you are in a python envrionment you use to build latex documents, then run:
```bash
git clone https://github.com/lstrsrmn/vehicle-lexer
cd vehicle-lexer
pip install -e .
```

To invoke the lexer, use the name `vehicle-lang`. (`vehicle` will conflict with the default one and cause problems, which using `vehicle-lang` avoids).
